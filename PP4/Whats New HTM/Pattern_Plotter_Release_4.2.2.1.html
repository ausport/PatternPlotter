What’s New in Pattern Plotter:

 

Version 4.2.2.1.

  1. ✓   New function added to clear all tag buttons.

  2. •  Right-click on the Tags Window and select "Clear All Tags" to activate.


  1. ✓    Video Playlist functions enhanced:

  2. •  Right-click to show selected VPL items in a Pathway Map.

  3. •  Option to select from existing playlists when appending new items.


  1. ✓    Interoperable charts and reports:

  2. •  Right-click on items in a Cluster Chart to transfer them into either a Video Playlist, or a Pathway Map.

  3. •  Right-click on items in a Video Playlist to transfer them into a Pathway Map.

  4. •  Right-click on an item in a Pathway Map to transfer it into a Video Playlist.


  1. ✓   Exportable Statistics:

  2. •  Right-click on the Statistics report to export the data into an Excel spreadsheet.

  3. •  Right-click on the Statistics report to switch between tabulation formats.


  1. ✓   Database portability:

  2. •  Import or export individual games, or collections of games between computers.
